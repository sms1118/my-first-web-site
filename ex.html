html의 구조

본문을 설명하는 부분은 [head] 라는 태그로 (예를 들어 title 태그와 meta 태그)
본문내용 부분은 [body] 라는 태그로 묶기로 함

그리고 [head] 와 [body] 를 [html] 이라는 태그로 또 묶음
또 '이 문서는 html 이다' 라는 뜻으로 [html] 태그 위에 [!doctype html] 이라고 씀

타이틀(웹 이름)

[title] WEB - html [/title]
title 은 웹의 이름을 바꿀 때 사용하는 태그임

meta(메타)

[meta charset="utf-8"] 이라는 명령어가 있음
charset 의 char 은 문자란 뜻이고 set 은 규칙 이라는 뜻임
utf-8 은 문서의 저장방식임
결과적으로 [meta charset="utf-8"] 은
'이 문서의 저장방식이 utf-8 으로 되어 있으니까 이거로 열어라' 라는 뜻임

링크 삽입

anchor(앵커) 의 첫글자를 딴 a 를 사용하고 링크를 표현할때 씀 특정 텍스트에 링크를 걸고싶을때 사용함

[a href="링크"]
먼저 a 태그를 쓰고 hypertext 의 첫 글자인 h 그리고 '이 웹브라우저를 참조해'라고 해서 reference 의 ref 를 사용

만약 링크의 웹페이지가 새탭으로 열리게 하고 싶으면
앞부분의 a 괄호 안에 [target="_blank"] 를 써주면 됨

또 링크를 클릭하기전에 이 링크가 무엇인지 알게하고 싶다면
[title="링크 설명"] 을 추가로 써주면 됨

    문단 나눔

    p 는 문단을 나눌때 사용함
    정하고자 하는 문단의 맨 앞과 맨 뒷부분에 p와 /p를 사용하면
    정해진 간격만큼 위아래로 띄어짐

    추가로 CSS에서 사용하는 언어
    stlye="margin-top or bottom:숫자px;"를 사용하면
    문단의 위 혹은 아랫부분을 선택하여
    원하는 픽셀만큼 간격조절이 가능함
    margin은 여백이라는 뜻임

    이미지 삽입

    'img'는 이미지 삽입에 필요한 태그임(이미지의 줄임말)
    그 옆에 'source=""(소스)or src=""'라는 태그를 붙혀줌
    소스는 삽입하는 이미지가 어떤 이미지 인지를 알려주는 문법임

    삽입 할 이미지를 웹파일이 있는 폴더에 다운로드 받아줌
    파일이름이 복잡하면 간단하게 바꿔주고
    'img src="이미지 이름.jpg"'이렇게 ""사이에 이미지이름.jpg를 적어줌
    그리고 이미지의 크기를 조절하기 위해 한칸 띄어주고'width="숫자or퍼센트"'를 적어줌

    이미지 삽입할때나 문단을 픽셀단위로 띄울때
    태그 말고도 추가로 적어준것을 속성이라고 함(영어로 Attribute)
    ex)stlye="margin-top: n px;" / src="image.jpg" / width="n"
    속성을 적을때는 위치나 순서는 상관 없음

    태그 자체로만은 정보가 부족할때
    이런 속성을 이용해서 더 많은 의미나 정보를 부가할 수 있게됨

    꿀팁

    1. ctrl(컨트롤)을 누른상태로 커서를 여러곳 클릭하면 여러줄에 태그를 동시에 입력할 수 있음
    2. 원하는 줄들을 드래그하고 tab(탭)을 누르면 한칸 안으로 들어감
    3. 사용하려는 태그를 괄호 없이 적고 엔터(Enter)를 누르면 태그의 열림과 닫힘이 자동으로 생성됨
    생성된 태그 사이에 내용만 입력해주면 됨
